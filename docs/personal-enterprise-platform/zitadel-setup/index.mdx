# Zitadel Setup

This guide walks you through setting up Zitadel, an open-source identity and access management (IAM) solution, using the Proxmox VE Community Helper Script.

## 1. What is Zitadel?

Zitadel is a modern, cloud-native identity platform that provides:
- Single Sign-On (SSO)
- OAuth 2.0 and OIDC support
- Multi-factor authentication
- User management
- API authentication and authorization

## 2. Prerequisites

**Default System Requirements:**
- 1 vCPU
- 1 GB RAM
- 8 GB Storage
